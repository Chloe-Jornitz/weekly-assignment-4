My Carreer

Carreer:

Software engineering

Description:

Software engineering uses both engineering principles and programming to build or modify computer software.
Software engineers:
  • - Analyze the requirements for a project and decide what the software must do
  • - Design and plan software solutions using diagrams, flow charts, or models
  • - Use different tools and programming languages to write code
  • - Test code to assure that it functions correctly
  • - Find issues in code and fix code so that it runs smoothly and efficiently
  • - Create reports to keep track of changes and developments to projects
  • - Work collaboratively with other software engineers
They may create:
  • - Computer programs
  • - Computer software
  • - Websites
  • - Computer games
  • - Applications
  • - Operating systems

Future possibilities:

Demand is increasing for software engineers as the need for application, website, and software development as well as the use of artificial intelligence is rising. Because of this, most of the job openings in this career will come from new positions created by the expansion of the software industry. Growing need for mobile applications, medical software, AI applications and cybersecurity are examples of factors that increase the demand for software engineers.
